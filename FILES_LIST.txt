ИНСТРУКЦИЯ ПО РАЗВЕРТЫВАНИЮ:
-----------------------------
1. Запустите команду: npm run buildstatic
2. Распакуйте архив Academ-static-website.zip
3. Загрузите ВСЕ файлы из папки Academ/ на ваш веб-сервер
4. Сохраните структуру папок (особенно assets/Fonts/ и assets/icons/)
5. Откройте index.html в браузере

КОМАНДА СБОРКИ:
---------------
npm run buildstatic

Эта команда:
- Копирует шрифты из assets/Fonts/ в Academ/assets/Fonts/
- Создает ZIP архив Academ-static-website.zip
- Готовит все файлы для загрузки на сервер

ВАЖНО:
- Все пути к ресурсам относительные
- Структура папок должна быть сохранена
- Файлы готовы к использованию без дополнительной сборки
